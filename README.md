### react설치

node.js 설치를 한다.

### 초기설정

폴더 생성 후 명령어 npm init react-app .
실행 npm run start

### 개발된 프로젝트 빌드하기

npm run build

### 빌드한거 로컬에서 실행

npx serve build

### vsc 꿀팁

다른 파일에서 쓰이는 함수이름을 변경하고 모두 변경하고 싶을 때 정의해놓은 곳의 함수이름에서 F2를 누르고 변경.

같은 이름을 가진 변수 조회 ctrl shift l
